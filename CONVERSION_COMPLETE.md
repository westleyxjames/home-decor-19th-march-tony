## How to Use

### 1. Development
```bash
# Simply open index.html in a browser
open index.html

# Or use a local server
python -m http.server 8000
# Then visit: http://localhost:8000
```

### 2. Production Deployment
```bash
# Upload all files to your web server root:
- index.html
- about.html
- services.html
- pricing.html
- contact.html
- thank-you.html
- privacy-policy.html
- terms-conditions.html
- disclaimer.html
- cookie-policy.html
- styles.css
- script.js
```

### 3. Form Integration
Currently forms log to console. To connect to a backend:

**Option A: FormSubmit.co**
```html
<form action="https://formsubmit.co/your@email.com" method="POST">
```

**Option B: Custom API**
```javascript
// In script.js, update the form handler
fetch('/api/contact', {
    method: 'POST',
    body: JSON.stringify(formData)
})
```
